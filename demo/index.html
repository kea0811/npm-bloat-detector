npm-bloat-detector

Find the 5 deps eating your bundle

A tiny CLI that walks your node_modules, measures the real on-disk size of every dependency, and ranks the heaviest — so you know exactly what to trim before it ships.

⚖️

Real bytes, not estimates

It reads the actual files on disk and resolves pnpm/npm install symlinks — no registry guesses, no rough approximations.

🔍

Diff against a baseline

Save a snapshot, then see the exact byte delta per dependency — red when it grows, green when it shrinks.

🚦

Budget gate for CI

Set --budget 50mb and it exits non-zero when your install blows past the limit — one line in your pipeline.

Install

$ pnpm add -g npm-bloat-detector # or run it once, no install: $ pnpm dlx npm-bloat-detector

Common uses

# the top 5 heaviest dependencies in the current project $ npm-bloat-detector # show the top 10, dev dependencies included $ npm-bloat-detector --top 10 --dev # fail CI when the install exceeds a budget $ npm-bloat-detector --budget 50mb # snapshot now, diff on the next run $ npm-bloat-detector --save .bloat.json $ npm-bloat-detector --baseline .bloat.json
